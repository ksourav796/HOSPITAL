Easy!Appointments

Easy!Appointments is a highly customizable web application that allows your patients to book appointments with you via the web. Moreover, it provides the ability to sync your data with Google Calendar so you can use them with other services.

Current Version 1.3.0

Support

If you encounter any problems when using Easy!Appointments you can search the official Google Group for answers. You might also need to create a new issue on the Google Code page in order to help the development progress.
